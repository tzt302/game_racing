# Racing Line Pro v2.4.0

2.4.0 将原计划中的 2.3.3—2.3.6 改进合并为一次正式更新，重点重构转向物理、
松油减速、个人最佳圈 Delta 和计时赛赛道限制规则。

## 更真实的转向与轮胎

- 输入现在请求横向加速度，再由车速、轴距、抓地和下压力反算前轮舵角。
- 固定机械舵角与轻微转向不足梯度取代旧的高速强制缩舵逻辑。
- 轮胎横向力按松弛长度逐步建立，转向和快速反打不会瞬间完成。
- 制动与驱动通过摩擦圆消耗可用横向抓地；轻微循迹刹车帮助旋转，重刹则降低转弯能力。
- 修正超过抓地极限后的侧滑方向。

## 驾驶与计时

- 加入随 RPM、挡位和油门开度变化的发动机制动，改善松油减速。
- Delta 只与玩家自己的个人最快圈逐点轨迹比较。
- 计时赛越过赛道限制后立即删除本圈，不会刷新个人最佳或 Delta 基准。
- AI 正赛仍采用三次赛道限制警告后罚时 5 秒的规则。
- 删除赛道/全场最快圈 HUD 与菜单参考圈显示。

## 声音与开发工具

- 删除全部运行时声音效果，发行包不包含 WAV、MP3、OGG 或无授权 V8 素材。
- 加入强化学习步骤档案与安全默认关闭的训练代码骨架；本版本没有运行训练或附带模型权重。

## 验证

- Windows 单文件 EXE 完成启动冒烟测试。
- PyInstaller 包内容检查确认不含音频素材。

下载 `RacingLinePro-v2.4.0.exe` 后可直接运行，无需安装 Python。
